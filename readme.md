## FE(feature-editor)

### 使用(Usage)

```shell
npm install
npm start
start http://127.0.0.1:31136
```

### 开发(development)

1. 安装依赖(install dependencies)

* [node](https://nodejs.org)
* [npm:fb-flo](https://github.com/facebook/fb-flo)
* [r.js](https://github.com/jrburke/r.js)

2. 开启开发模式(begin dev-mode)

```shell
npm run-script dev
```
